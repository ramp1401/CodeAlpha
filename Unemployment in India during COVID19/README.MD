# 📊 Unemployment Rate Analysis & Prediction

## 📌 Project Overview
This project focuses on analyzing unemployment data to uncover trends and build a predictive model for unemployment rate.

---

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Handle missing values and outliers
- Analyze unemployment trends across regions and time
- Compare urban vs rural unemployment
- Build a machine learning model to predict unemployment rate

---

## 🛠️ Tech Stack
- Python 🐍
- Pandas
- NumPy
- Matplotlib & Seaborn
- Scikit-learn

---

## 📂 Dataset Features
- Region
- Date
- Frequency
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)
- Area (Urban/Rural)

---

## 🧹 Data Cleaning
- Removed null values
- Dropped empty rows
- Handled outliers using IQR method
- Converted Date column into year and month

---

## 📊 Exploratory Data Analysis (EDA)
- Unemployment rate by region
- Urban vs rural comparison
- Time-based trend analysis
- Outlier visualization using boxplots

---

## 🤖 Machine Learning
- Model: Random Forest Regressor
- Target: Estimated Unemployment Rate (%)

### Evaluation Metrics:
- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)

---

## 📈 Key Insights
- Certain regions have consistently higher unemployment rates
- Urban and rural areas show noticeable differences
- Time-based trends highlight economic changes

---
